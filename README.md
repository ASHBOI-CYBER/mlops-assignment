# MLOps Housing Prediction Pipeline

## Project Overview
This project orchestrates an end-to-end Machine Learning pipeline using **MLflow Projects** and **DVC**.

## How to Run
1. Install dependencies: `pip install -r requirements.txt`
2. Run pipeline: `mlflow run . --env-manager=local`

## Features
* **Data Versioning:** DVC
* **Orchestration:** MLflow
* **CI/CD:** GitHub Actions
